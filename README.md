# XML Course Exercises

A comprehensive repository for XML course assignments and projects from FMI (Faculty of Mathematics and Informatics).

## Project Overview

This repository contains XML-based exercises including:
- **XML Schema Definition (XSD)** - Schema validation rules
- **XML Documents** - Structured data in XML format
- **XSLT Transformations** - XSL stylesheets for XML transformation
- **DTD Definitions** - Document Type Definitions
- **CSS Styling** - Visual presentation of XML content
- **Images** - Supporting media assets
